ABOUT US

BMSCE Events

BMSCE Events is a website that contains all the information about the extra-curricular activities happening in our college. This website is exclusive for BMSCE members only. The website requires login id to be accessed by the user after which the user is directed to the main page from where he/she can navigate to the four sections of the website.All the students can go to the respective sections and register for the event they are interested in by clicking of the registration form linked to each event. The user can even pay and attach the screenshot of the payment with the form itself. Their data is stored in the database and then the respective in-charge can check the backend for the information. This method eliminates scope for error and makes the process of registration hassle free.
